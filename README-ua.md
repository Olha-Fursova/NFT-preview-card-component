### Опис завдання

Ваше завдання — створити цей компонент картки попереднього перегляду та зробити його максимально наближеним до дизайну.

Ви можете використовувати будь-які інструменти, які вам подобаються, щоб допомогти вам виконати завдання. Тож, якщо у вас є щось, що ви хотіли б потренуватися, сміливо спробуйте.

Ваші користувачі повинні мати можливість:

- Переглядати оптимальний макет залежно від розміру екрана свого пристрою
- Переглядати стани наведення курсора для інтерактивних елементів

### Використані технології

- HTML5 для розмітки
- CSS3 стилізація
- Flexbox
